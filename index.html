交通银行悉尼分行 举报材料 第一部分 内部文件

您好，我希望通过此匿名邮箱举报交通银行悉尼分行管理层存在的多项涉嫌非法行为及职场霸凌问题。这些问题严重影响了分行的工作环境、员工权益以及公司的声誉。以下是详细内容：

一、分行行长夏慧昌（生日1968年1月2日）的贪腐、失职与不当行为
1. 纵容和参与系统性腐败与裙带关系
夏慧昌对悉尼分行的招聘与提拔过程中存在的裙带关系、腐败现象负有不可推卸的责任，例如对路易凝（1992年5月11日）、马雪侨（1996年6月13日）、宋晗（1982年2月13日），何涛（1979年5月16日），潘昊伟（1982年6月15日），吴志坤（1977年5月20日），李昕庆（1974年10月13，无任何招聘面试记录）等员工的录用和提拔。
2. 参与与纵容采购回扣
在采购酒水的过程中，夏慧昌与公司部主管陈金锟被指控收取16万澳币的回扣。尽管成立了内部调查小组，但夏慧昌利用职务之便干预调查，掩盖事实真相。（附件一 酒类采购）
3. 失职导致内部管理混乱
夏慧辉昌未能及时制止陈金锟、李宣（1971年11月4日）、宋晗、潘昊伟、纪英（1976年4月20）、苏榆（1978年9月4日）等管理层和部门主管的职场霸凌与违规行为，任由分行内部秩序混乱、士气低落。这期间作为分行COO的黄少辉（Alan Huang）和HR主管吴志坤在夏慧昌的授意下不作为。
4. 滥用职权招聘和推动不合格员工提拔
夏慧昌多次利用其职务权力，为能力不足但与其关系密切的员工提供提拔机会，助长了分行的不正之风。也有人谣传布里斯班分行夏冰（生日1986年11月4日）是夏慧昌直系亲属。
5. 不当私生活行为
有员工多次在周末加班时，看到夏慧昌带不同年轻女性出入办公室，举止亲昵。此外，也有员工声称曾被要求周末不能来加班，因为办公室被夏慧昌占用。夏辉昌的房间内还违规安装了可以上网的电脑，用于炒股。

二、陈金锟的问题汇总
陈金锟在悉尼分行的行为严重违反职业道德和公司规章制度，涉及触犯澳洲刑法，但依然没有受到相应处罚，具体问题包括：
1. 采购回扣与资金违规
在酒类采购中被指控收取16万澳币回扣，部分资金通过沈腾等人虚报发票，并转为现金，给与陈金锟，形成隐蔽资金转移链条。分行内部调查避重就轻，有筛选性选择交易，让其躲过追查。（附件一 酒类采购）
主导办公室搬迁及装修项目，涉及金额高达500-700万澳币，缺乏透明度，涉嫌利益输送。（请相关部门对项目和成本进行仔细核算，尤其是涉及陈金锟主导的项目）
2. 职场暴力与霸凌
多次辱骂、威胁下属，最严重一次掐住同事王一俊脖子，被同事看到；另外一次喊着“弄死CRO”，并且撞CRO办公室门（严重刑事犯罪）；对下属和一般同事辱骂更是家常便饭，严重破坏职场环境。这在澳洲是严重违规，并涉及刑事犯罪，但都被夏慧昌掩盖下来。
3. 裙带关系与性贿赂
陈金锟利用与总行副行长的关系规避举报，获得台北分行副行长提拔。其自己和其下属吃饭时经常炫耀其和总行副行长的关系，并说因此升迁。
据悉，李青（河南籍）通过对陈金锟（河南籍）提供性贿赂，获得了岗位上的优待和职业晋升机会。（公司部有多位河南籍员工，请彻查）
4. 权力滥用与团队氛围破坏
控制李青（河南籍）、沈腾（河南籍）的升迁和奖金分配，强化对下属的权力掌控，破坏分行内部公平环境和团队氛围。沈腾在陈金锟收取回扣过程中充当白手套。（附件一 酒类采购）

三、招聘问题，涉及系统性腐败
1. 财务部路易凝的招聘背景
路易凝毕业于一所二流大学，英语沟通能力和业务能力极差，其招聘背景涉嫌通过性贿赂或其他方式获得职位，其能力与财务岗位要求严重不符，涉嫌性贿赂和贿赂。
2. 公司部马雪侨的招聘问题
马雪侨曾在光大悉尼分行短暂任职，后通过不明渠道进入交行悉尼分行。
光大悉尼分行现任行长为原交行悉尼分行行长，与夏慧昌关系密切。马雪侨的录用背景存在通过性贿赂、贿赂或裙带关系获得职位的可能性。
3. 资金部宋晗的升职问题
宋晗的升职背景可能与其妻的家庭关系有关，其妻为交通银行重庆分行现任或前任行长的女儿。其人飞扬跋扈，目中无人，经常霸凌和欺负同事（办公室掀翻桌子、强迫报销、辱骂同事和领导等）。
4. 裙带关系泛滥
招聘中普遍存在裙带关系和性贿赂现象，严重破坏了分行内部的公平竞争环境。
5. 招聘过程由皮包公司主导
为方便招聘的系统性腐败，招聘过程的主要是由一家皮包公司承担，该公司涉嫌在招聘过程中进行内部选人，并通过利益输送获取数百万澳币。此类做法严重影响了招聘的公正性，并导致了与公司要求严重不符的人员进入岗位，进一步加剧了内部腐败现象。（附件2 The Baygroup）

四、提拔任用问题
1. 外派员工的优先提拔
外派人员中，如公司部潘昊伟、内审部何涛、HR吴志坤等，经常被优先考虑提拔，尽管这些人素质低下，业务能力严重不足，英语沟通能力奇差，完全不能履行职责。
2. 360考核机制的漏洞
提拔机制以360考核为主，但因评分不公（部分人多部门主导）和主管评价主观，考核成绩严重失真，成为夏慧昌按个人喜好任用员工的借口。

五、公司部问题
1. 公司部员工沈腾、李青、肖飞的问题
o 沈腾： 帮助陈金锟违规报销，获取资金，并涉及对女同事性骚扰，多次尾随女同事入电梯，假装偶遇，并随至车站，并在短信和言语上对女同事进行性骚扰。
o 李青： 涉及性贿赂和帮助陈金锟在采购上违规。
o 肖飞（1981年8月20日）： 经常在私下谈论公司女同事的身材外貌等不当言论，并参与陈金锟的违规采购，尽管行为不当，但仍被升职。
这些问题表明公司部存在严重的职业道德缺失，并且管理层未采取有效措施进行制止。

六、外派人员问题
外派人员在交通银行悉尼分行担任了多个关键岗位，且这些外派人员普遍存在能力不足、管理不当的问题。具体问题如下：
1. 何涛： 英语能力极差，无任何内审业务能力，经常在上班时间做代购，后因拍夏慧昌马屁，调入分行内审部担任内审部主管，尽管其并不具备审计知识，英语能力完全无法达到工作的基本要求。
2. 吴志坤： 素质低劣，能力低下，不具备一般的英语沟通能力和人力资源知识和资质，作为HR主管，涉嫌滥用职权，对招聘、提拔、薪酬分配等方面存在偏私行为，屡次违反公司规章制度。并且严重违规泄露人事案件中的隐私信息，夏慧昌对此知之任之。
3. 李宣（副行长）： 存在多次职场霸凌行为，言辞恶劣，工作方式粗暴，不尊重下属，导致其管辖部门士气低落。
4. 外派人员的整体问题： 外派人员整体素质低下，工作能力严重不足，且普遍存在违反公司规章制度的行为，夏慧昌未能进行有效管理和监督。大部分外派人员不具备在澳洲履行其职责的能力，但都被夏慧昌录用提拔。

七、请求
1. 派遣独立小组，全面调查陈金锟和夏慧昌的贪腐及性贿赂、贿赂问题
包括采购回扣、办公室装修项目中的资金流向、职场暴力行为以及设计与女员工的性贿赂问题。
2. 全面调查招聘背景与裙带关系
重点审查路易凝和马雪侨等人的招聘背景，查明是否通过性贿赂或其他违规手段获得职位。
3. 审查分行提拔机制的合理性
针对360考核机制和主管评价主观性问题进行全面审查，确保提拔过程公平、公正、透明。
4. 强化分行管理监督与外派人员考核
针对分行管理层和外派人员加强监督，确保其符合公司规章和职业道德标准，杜绝裙带关系和权力滥用。

以上举报内容仅为若干前员工及现员工的总结与归纳，所揭露的只是部分问题，更多的违规行为尚待进一步揭示。该中资行海外分行的管理存在诸多不规范行为，若此类问题持续发生，不仅会损害企业声誉，更可能影响到国家形象及国际信誉，望相关部门予以高度关注。
